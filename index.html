Internet

What is the Internet?

The Internet is a global system that connects millions of computers, smartphones, tablets, servers, and other digital devices around the world. It allows these devices to communicate with each other and share information. The word "Internet" comes from the term "Interconnected Network," which means many networks connected together. It is not a single thing or a single website, but a huge network made up of many smaller networks connected globally. This is why the Internet is often called the “network of networks.”

Imagine that every computer and smartphone in the world is connected by invisible roads. These roads allow information to travel from one device to another. The Internet is like a huge transportation system for information. Instead of carrying people or goods, it carries data such as text, images, videos, emails, and files.

The Internet allows information to move from one place to another in a very short time. It works using physical infrastructure like fiber-optic cables, wireless signals, satellites, and routers that help data travel across the world. In daily life, the Internet is used for communication, education, entertainment, business, and many services. Without the Internet, modern digital communication and online services would not exist.

How the Internet Works

    The working of the Internet is based on three main concepts:


  1. the Client-Server Model

  2. HTTP (HyperText Transfer Protocol)

  3. DNS (Domain Name System)

These work together to deliver information quickly and efficiently.



Client-Server Model

The Client-Server model is the basic structure of how communication happens on the Internet. When a user opens a website, the client sends a request to the server. The server processes the request and sends back the required data. The client then displays the information on the screen.

A client is the device you use.It is used to request information
The device such as:

  • Mobile phone
  • Laptop
  • Computer

A server is a powerful computer that stores that stores websites, applications, and data
Example:

  • Google has servers
  • youtube has servers
  • Facebook has servers

    • etc..

HTTP (HyperText Transfer Protocol)

HTTP is the communication rule used between the client and server. It defines how messages are formatted and transmitted, and how web servers and browsers should respond to different requests. Without HTTP, computers cannot understand each other on the web.

Simple Meaning:

HTTP is the language used for communication between browser and server

DNS (Domain Name System)

DNS is a system that translates human-readable website names into machine-readable IP addresses. Computers do not understand names like “google.com”. They understand numbers called IP addresses. DNS solves this problem.

Simple Meaning:

DNS = converts website names into IP addresses

Step-by-Step Working of the Internet


  1. User enters a website name :

    2. The user types a website address (e.g., www.google.com) in a web browser.


  2. Browser sends a request to DNS :

    3. The browser asks the DNS server to find the IP address of the website.


  3. DNS finds the IP address :

    4. DNS converts the domain name (google.com) into an IP address that computers can understand.


  4. Browser connects to the server :

    5. Using the IP address, the browser finds and connects to the correct web server.


  5. Browser sends an HTTP request :

    6. The browser requests the webpage from the server using HTTP.


  6. Server processes the request :

    7. The server receives the request and finds the required webpage and resources.


  7. Server sends an HTTP response :

    8. The server sends the webpage data (HTML, CSS, images, etc.) back to the browser.


  8. Browser displays the website :

    9. The browser receives the data and displays the webpage on the user's screen.


Main Services on the Internet

The Internet provides many services that help people communicate, learn, work, share information, and entertain themselves. These services make the Internet useful in our daily lives.

  • World Wide Web (WWW)

    The World Wide Web (WWW) is one of the most popular Internet services. It allows users to access websites and web pages using a web browser.

    Examples:

    • Searching for information
    • Reading news
    • YouTube

    Uses:

    • Google
    • Wikipedia
    • Visiting websites

  • Email (Electronic Mail)

    Email is a service that allows people to send and receive messages electronically through the Internet.

    Examples:

    • Gmail
    • Yahoo Mail
    • Outlook

    Uses:

    • Sending messages
    • Sharing documents
    • Business communication

  • File Transfer Protocol (FTP)

    FTP is a service used to transfer files from one computer to another through the Internet.

      Uses:

    • Uploading files to websites
    • Downloading files from servers
    • Sharing large files

  • Social Media

    Social media allows people to communicate, share information, photos, and videos with others.

    Examples:

    • Facebook
    • Instagram
    • X (Twitter)
    • TikTok

    Uses:

    • Connecting with friends and family
    • Sharing content
    • Following news and trends

  • Video Streaming

    Video streaming allows users to watch videos online without downloading them completely.

    Examples:

    • YouTube
    • Netflix
    • Disney+

    Uses:

    • Watching movies
    • Watching educational videos
    • Live streaming events

  • Cloud Storage

    Cloud storage allows users to store files online and access them from anywhere with an Internet connection.

    Examples:

    • Google Drive
    • OneDrive
    • Dropbox

    Uses:

    • Storing documents
    • Backing up files
    • Sharing files online

  • Online Banking

    Online banking allows customers to perform banking activities through the Internet.

    Uses:

    • Checking account balances
    • Transferring money
    • Paying bills
    • Managing bank accounts

  • Video Conferencing

    Video conferencing allows people to communicate face-to-face through video calls over the Internet.

    Examples:

    • Zoom
    • Google Meet
    • Microsoft Teams

    Uses:

    • Online classes
    • Business meetings
    • Virtual interviews

  • Online Shopping (E-Commerce)

    E-commerce allows users to buy and sell products and services online.

    Uses:

    • Purchasing products
    • Ordering food
    • Booking tickets
